Это реализация сервера(Servant) для тестового задания для АРМО.
Тестовое задание:
С помощью Qt (с++11) написать кроссплатформенное распределенное приложение позволяющее
обмениваться данными с помощью технологии CORBA.
Пример предоставить для ОС Windows.
Распределенное приложение должно работать на двух узлах сети с ОС Windows и выполнять следующее: на одном узле загружаем произвольное изображение, передаем его с помощью CORBA на другой узел и отображаем на нем. Допустимо использование GUI.

Изображение отображается на этом сервере. Реализация клиента находится тут.
